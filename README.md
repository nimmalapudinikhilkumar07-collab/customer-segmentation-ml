Customer Segmentation using K-Means

📌 Project Overview

This project performs customer segmentation based on Annual Income and Spending Score using K-Means clustering.

🎯 Objective

To group customers into different segments so businesses can target them effectively.

🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-learn

⚙️ Steps Performed

- Data cleaning and preprocessing
- Feature selection
- Data scaling using StandardScaler
- Finding optimal clusters using Elbow Method
- Applying K-Means clustering
- Assigning customer types based on clusters

📊 Output

Customers are divided into 5 segments:

- Low Value Customers
- Premium Customers
- Average Customers
- Impulse Buyers
- Target Customers

🚀 Result

Helps businesses understand customer behavior and apply targeted marketing strategies.
